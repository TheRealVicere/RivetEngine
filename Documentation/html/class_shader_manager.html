static void ShaderManager::Load ( const std::string &  key,
const std::string &  path 
)
static

Same as PrivLoad(key, path), but accessible to users through the resource loader source file.

Parameters
keyThe string key used to return the shader.
pathRelative pathname of the shader file. Starts in the project's Assets\Shaders folder. Note that the path should not contain the file extension.

The resource loader contains every shader for user convenience. Comment out those not needed, and use the rest with whatever keys the user deems appropriate.
