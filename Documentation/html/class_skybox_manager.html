static void SkyboxManager::Load ( const std::string &  key,
const std::string &  path,
float  len = 30.0f 
)
static

Same as PrivLoad(key, path), but accessible to users through the resource loader source file.

Parameters
keyThe string key used to return the skybox.
pathRelative pathname of the skybox's texture file. Starts in the project's Assets\Textures folder.
len(Optional) The length of the skybox's dimensions.

Choosing the correct length may take some trial and error. We think 30 is a good size, hence why it's our default length.

Note that all textures MUST be in the .tga format.
