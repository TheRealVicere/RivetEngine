◆ SetCollisionPair()

template<typename C1 , typename C2 >
template< typename C1, typename C2 > void CollisionManager::SetCollisionPair ( )
inline

Enables two types of objects to process collision with objects of each other's type.

Template Parameters
C1The first object type.
C2The second object type.
Example:
Say you have two objects of types Player and Enemy that you want to process collision for. Set up their collision like this:
SetCollisionPair<Player, Enemy>();
Then, make sure to include corresponding functions for collision processing:
void Player::Collision(Enemy* e) { ... }
void Enemy::Collision(Player* p) { ... }
Note that C1 and C2 are interchangeable and do not require separate function calls:
// These two do the same thing!
SetCollisionGroup<C1, C2>();
SetCollisionGroup<C2, C1>();
