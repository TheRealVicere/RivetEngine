GodCam::GodCam ( Camera *const  mCam)

Constructs a new god camera.

Parameters
[in,out]mCamThe main camera.
Example:
This constructor passes the camera used for gameplay purposes since turning off the god camera disables the new camera entirely. This allows the camera manager to reuse the main gameplay camera when the god camera is not in use. Therefore, unless the user wishes to have more than one perspective camera in a scene, the best use of this function is as follows:
GodCam gc(SceneManager::GetCurrent()->GetCurrent3DCamera());
