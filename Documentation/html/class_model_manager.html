◆ Load() [2/2]

static void ModelManager::Load ( const std::string &  key,
const std::string &  path 
)
static

Same as PrivLoad(key, path), but accessible to users through the resource loader source file.

Parameters
keyThe string key used to return the model.
pathRelative pathname of the model file. Starts in the project's Assets\Models folder.
Example:

The following is an example of how to store an arbitrary model:

ModelManager::Load("My Model", "CoolModel.azul");

Note that all models MUST be in the .azul format. A converter from .fbx to .azul will be included in the first release and onward for user convenience.
