Stores a terrain for later use. The given key is used to return the stored terrain. Note that keys must be unique.

Parameters
keyThe string key used to return the terrain.
heightmapRelative pathname of the terrain heightmap file. Starts in the project's Assets\Textures folder.
texThe texture used on the terrain.
lenThe length of the terrain, in AZUL units.
maxheightThe height of the terrain model.
ytransThe Y offset used to place the terrain.
RepeatUThe number of times to use the texture's UV coordinates along the U axis.
RepeatVThe number of times to use the texture's UV coordinates along the V axis.
