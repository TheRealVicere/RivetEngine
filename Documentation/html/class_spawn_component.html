◆ EnterScene()

void SpawnComponent::EnterScene ( )
privatevirtual

Used to enable functionality of an object.

All of an object's components should be enabled here.

Example:
Suppose a game object MySpawnComponent needs to respond to pressing the Enter key and needs update, draw, and collision functionality. This can be implemented as follows:
void MySpawnComponent::EnterScene()
{
EnableUpdates();
EnableDrawing();
EnableCollision();
AddInput(AZUL_KEY::KEY_ENTER, KeyboardManager::EventType::KeyPressed);
}

◆ ExitScene()

void SpawnComponent::ExitScene ( )
privatevirtual

Used to disable functionality of this object.

All inputs and alarms should be cleared here. Collision, updates, and drawing should also be disabled here if needed.

Example:
Suppose a game object MySpawnComponent was responding to pressing the Enter key and used update, draw, and collision functionality. Safely disabling these can be implemented as follows:
void MySpawnComponent::EnterScene()
{
DisableUpdates();
DisableDrawing();
DisableCollision();
ClearInput(AZUL_KEY::KEY_ENTER, KeyboardManager::EventType::KeyPressed);
}
